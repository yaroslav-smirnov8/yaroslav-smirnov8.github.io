# Professional Portfolio - Python Backend Engineer

A clean, professional GitHub Pages portfolio showcasing production-grade systems built with Python, AI integration, and reliable architecture.

## Portfolio Structure

```
portfolio/
├── index.md                 # Main landing page
├── about.md                 # About & contact page
├── skills.md                # Technical skills summary
├── deployment-guide.md        # GitHub Pages deployment guide
├── README.md               # This file
├── assets/
│   ├── css/
│   │   └── style.md      # Custom styling (CSS in markdown)
│   └── images/             # Project screenshots/architecture diagrams
└── projects/
    ├── index.md             # Projects overview page
    ├── invbot/             # EvoTeach AI Bot
    ├── matrix-quest/       # Matrix Teaching Quest
    ├── platform-teachers/   # AI Course Generator Platform
    ├── store/              # AI Store Platform
    ├── teacherhelper/       # Teacher Scheduler Bot
    └── train-prompts-bot/  # Train Prompts Bot
```

## Featured Projects

1. **EvoTeach AI Bot** - Production-grade AI platform serving 2,847 active users with 99.9% uptime
2. **Matrix Teaching Quest** - Interactive quest system with first-party analytics and conversion funnels

## Deployment

Follow the step-by-step guide in [deployment-guide.md](deployment-guide.md) to deploy this portfolio to GitHub Pages.

## Customization

To customize this portfolio for your own use:

1. Update personal information in `about.md`
2. Replace placeholder links with your actual contact information
3. Add your own projects following the existing structure
4. Update the `_config.yml` file with your name and details
5. Add your own project screenshots to `assets/images/`
